============
Installation
============


Stable release
--------------

To install Apple Icon Set Generator, run this command in your terminal:

.. code-block:: console

    $ pip install icon_set

This is the preferred method to install Apple Icon Set Generator, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Apple Icon Set Generator can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/s16h/icon_set

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/s16h/icon_set/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/s16h/icon_set
.. _tarball: https://github.com/s16h/icon_set/tarball/master
